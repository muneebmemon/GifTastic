# Giphy Display App (using AJAX)

The app named GifTastic has to do with getting specific themed data (Animal's GIFs) from GIPHY API using AJAX & JQuery and then displaying it on to the webpage.

## Technologies used
 - JQuery
 - HTML
 - Javascript
 - AJAX
 - CSS
 - Giphy API
